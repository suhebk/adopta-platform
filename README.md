# Adopta Platform

Adopta is a production-grade, multi-tenant SaaS Digital Adoption Platform.

It provides:
- In-app guidance
- Walkthroughs
- Smart tips
- Checklists
- Resource centre
- Authoring studio
- Analytics
- Segmentation
- AI help
- Governance
- Enterprise SaaS security

This repository is separate from Property Income MTD.

Property Income MTD integration will be delivered later through a first-party Adopta runtime SDK and safe adoption context contract.

See `docs/adopta/` for the full specification pack.
